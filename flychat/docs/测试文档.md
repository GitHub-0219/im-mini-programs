# FlyChat 测试文档

## 1. 测试用例

### 1.1 消息列表页（index.html）

| 编号 | 测试项 | 预期结果 |
|------|--------|----------|
| TC-001 | 页面加载 | 显示会话列表，包含头像、昵称、最后消息、时间 |
| TC-002 | 搜索功能 | 输入关键词后实时过滤会话列表 |
| TC-003 | 置顶标记 | 置顶会话显示📌标记，排在列表前面 |
| TC-004 | 免打扰标记 | 免打扰会话显示🔇标记，不显示未读角标 |
| TC-005 | 未读角标 | 未读数>0时显示红色角标 |
| TC-006 | 点击会话 | 跳转到对应的聊天页面 |
| TC-007 | 底部Tab | 点击切换消息/通讯录/工作台/我 |

### 1.2 聊天页（chat.html）

| 编号 | 测试项 | 预期结果 |
|------|--------|----------|
| TC-008 | 消息气泡 | 自己的消息蓝色靠右，对方白色靠左 |
| TC-009 | 文字消息 | 正确显示文字内容 |
| TC-010 | 文件消息 | 显示文件图标、名称、大小 |
| TC-011 | 语音消息 | 显示语音图标和时长 |
| TC-012 | 消息状态 | 已读显示✓✓（蓝色），已送达显示✓✓（灰色） |
| TC-013 | 发送消息 | 输入文字点击发送，消息出现在聊天区域 |
| TC-014 | Enter发送 | 按Enter键发送消息 |
| TC-015 | Shift+Enter | 换行而不发送 |
| TC-016 | @功能 | 输入@弹出成员列表，选择后插入 |
| TC-017 | 功能面板 | 点击+号展开/收起功能面板 |
| TC-018 | 右键菜单 | 长按/右键消息弹出菜单（回复/转发/复制/撤回） |
| TC-019 | 消息引用 | 点击回复，输入框显示引用信息 |
| TC-020 | 返回按钮 | 移动端点击返回上一页 |
