# FlyChat 小程序适配说明

## 1. 概述

FlyChat 当前为纯 HTML+CSS+JS 的 Web 应用。本文档说明如何将其适配到微信小程序、飞书小程序和钉钉小程序平台。

## 2. 通用适配方案

### 2.1 代码结构映射

| Web | 微信小程序 | 飞书小程序 | 钉钉小程序 |
|-----|-----------|-----------|-----------|
| HTML | WXML | FXML | AXML |
| CSS | WXSS | FSS | ACSS |
| JS | JS (小程序API) | JS (飞书API) | JS (钉钉API) |
| 模板语法 | {{}} / wx:if | {{}} / tt:if | {{}} / a:if |

### 2.2 目录结构转换

```
mini-program/
├── app.js              # 应用入口
├── app.json            # 应用配置
├── app.wxss / .fss / .acss  # 全局样式
├── pages/
│   ├── index/          # 消息列表
│   │   ├── index.wxml
│   │   ├── index.wxss
│   │   └── index.js
│   ├── chat/           # 聊天
│   ├── contacts/       # 通讯录
│   ├── workspace/      # 工作台
│   ├── profile/        # 个人中心
│   └── group/          # 群组管理
├── components/         # 公共组件
│   ├── conv-item/      # 会话项

```

## 6. 适配注意事项

### 6.1 通用注意事项

1. **DOM操作**：小程序不支持 `document`、`window`，需用数据绑定替代
2. **事件绑定**：`onclick` → `bindtap`，`oninput` → `bindinput`
3. **条件渲染**：`display:none` → `wx:if` / `tt:if` / `a:if`
4. **列表渲染**：`innerHTML` → `wx:for`
5. **样式单位**：`px` → `rpx`（1rpx = 0.5px）
6. **页面通信**：URL参数 → `eventChannel` / 全局状态

### 6.2 平台差异处理

```javascript
// 平台判断
const platform = {
  isWechat: typeof wx !== 'undefined' && wx.getSystemInfoSync,
  isFeishu: typeof tt !== 'undefined' && tt.getSystemInfoSync,
  isDingtalk: typeof dd !== 'undefined' && dd.getSystemInfoSync
};

// 统一API封装
function showToast(title) {
  if (platform.isWechat) wx.showToast({ title, icon: 'none' });
  else if (platform.isFeishu) tt.showToast({ title, icon: 'none' });
  else if (platform.isDingtalk) dd.showToast({ content: title });
}
```

### 6.3 性能优化

1. 使用分包加载减少首屏时间
2. 长列表使用虚拟列表（recycle-view）
3. 图片使用CDN + 懒加载
4. 减少 `setData` 调用频率，合并数据更新
